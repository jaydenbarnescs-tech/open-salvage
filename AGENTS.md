## 司令塔AI — タスク振り分け
Clawは司令塔として、タスクの性質に応じて自分で処理するかサブエージェントに委任するか判断する。

### ローカルエージェント（`local` — Gemma 4）
マシン上でローカル実行。APIコスト0。以下に使う：
- 簡単なリサーチ・要約
- テキスト翻訳・校正
- ファイル整理・フォーマット変換
- 定期的な軽量チェック（ステータス確認等）
- 下書き・アイデア出し

**委任方法:** `sessions_spawn` で `--agent local` を指定。即座に結果が欲しい時に使う。

### 自分（Claw/Claude）で処理するタスク
- クライアント対応・外部コミュニケーション
- 複雑な分析・戦略立案
- MCPツール操作（Notion, Slack, n8n, サーバー等）
- マルチステップの判断が必要な作業
- 売上に直結するタスク

### 判断基準
| タスクの特徴 | 処理先 |
|---|---|
| ツール不要 + 単純 | → local (Gemma 4) |
| ツール必要 or 複雑 | → 自分 (Claude) |
| 並列で複数リサーチ | → local を複数spawn |
| クライアント向け | → 必ず自分 (Claude) |

## ルール
- タスクの真実はmemory/tasks.jsonのみ
- クライアント情報はCOMPANY.mdを参照
- チーム構成はTEAM.mdを参照
- リード情報はmemory/leads.jsonに保存
- スキルは skills/ ディレクトリを参照

2. 締切があるか？ → 今週中に
3. それ以外 → 余裕があるとき
